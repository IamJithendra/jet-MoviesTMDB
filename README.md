<h1 align="center"> 
   <a>Movies TMDB</a>
</h1>

An application that allows you to browse the content of The Movie Database (TMDB) created with Jetpack Compose


## Technologies
Project is created with:
* Jetpack Compose
* Accompanist
* Coroutines and Flow
* Hilt
* Paging Compose
* OkHttp
* Retrofit
* Moshi
* Room
* Compose Destinations
* Coil
* MVVM
